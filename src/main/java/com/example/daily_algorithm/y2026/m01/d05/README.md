# 🧩 House Robber

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** Dynamic Programming  
**링크:** https://leetcode.com/problems/house-robber/

---

## 📝 문제 설명

도둑이 길을 따라 늘어선 집들을 털려고 한다.  
각 집에는 일정 금액의 돈이 들어 있다.

하지만 **인접한 두 집을 연속으로 털면 경보가 울린다.**

각 집에 있는 돈이 배열 `nums`로 주어질 때,  
경보를 울리지 않고 털 수 있는 **최대 금액**을 구하라.

---

## 🔍 예시

입력  
nums = [1,2,3,1]

출력  
4

설명  
1번 집(1) + 3번 집(3) = 4

---

입력  
nums = [2,7,9,3,1]

출력  
12

설명  
1번 집(2) + 3번 집(9) + 5번 집(1) = 12

---

## 📋 제약 조건 (Constraints)

- `1 <= nums.length <= 100`
- `0 <= nums[i] <= 400`

---

## 💡 접근 방식

- 현재 집을 털면 → 이전 집은 털 수 없다
- 현재 집을 안 털면 → 이전까지의 최대 금액을 그대로 유지

지금 방문한 집(전전 집 + 지금 집)을 터는 게 이득인지, 털지 않는 게(전 집) 이득인지 파악하기
