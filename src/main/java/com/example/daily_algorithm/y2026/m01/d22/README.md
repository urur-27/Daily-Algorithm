# 🧩 Valid Parentheses

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** Stack, String  
**링크:** https://leetcode.com/problems/valid-parentheses/  

---

## 📝 문제 설명

문자열 `s`가 주어질 때, 괄호 문자열이 **유효한지** 판단하라.

유효한 괄호 문자열 조건:
- 여는 괄호는 같은 종류의 닫는 괄호로 닫혀야 한다.
- 여는 괄호는 올바른 순서로 닫혀야 한다.
- 모든 닫는 괄호는 대응되는 여는 괄호가 존재해야 한다.

괄호 종류는 `()`, `{}`, `[]` 이다.

---

## 🔍 예시

입력  
s = "()"

출력  
true

---

입력  
s = "()[]{}"

출력  
true

---

입력  
s = "(]"

출력  
false

---

입력  
s = "([])"

출력  
true

---

## 📋 제약 조건 (Constraints)

- `1 <= s.length <= 10^4`
- `s`는 오직 `()[]{}` 문자로만 구성된다.

---

## 💡 접근 방식


괄호는 “가장 마지막에 열린 괄호가 가장 먼저 닫혀야” 한다.  
즉, LIFO 구조가 필요하므로 Stack이 적합하다.

### 진행 방식
- 문자를 왼쪽부터 순회한다.
- 여는 괄호 `(`, `{`, `[` 는 스택에 push 한다.
- 닫는 괄호 `)`, `}`, `]` 를 만나면:
  - 스택이 비어있으면 대응할 여는 괄호가 없으므로 false
  - 스택의 top이 해당 닫는 괄호와 짝이 맞는지 확인
  - 맞으면 pop, 아니면 false
- 순회가 끝났을 때 스택이 비어있으면 true (모두 정상적으로 닫힘)
