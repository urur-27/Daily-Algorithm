# 🧩 Longest Consecutive Sequence

**난이도:** Medium  
**유형:** Array, Hash Table (HashSet)  
**링크:** https://leetcode.com/problems/longest-consecutive-sequence/

---

## 📝 문제 설명

정수 배열 `nums`가 주어졌을 때, **연속된 숫자들로 이루어진 가장 긴 길이**를 반환하라.

- 연속 수열은 `x, x+1, x+2, ...` 처럼 값이 1씩 증가해야 한다.
- 배열은 정렬되어 있지 않을 수 있다.
- 시간 복잡도 **O(n)** 으로 해결해야 한다.

---

## 🔍 예시

**예시 1**
- 입력: `nums = [100,4,200,1,3,2]`
- 출력: `4`
- 이유: 가장 긴 연속 수열은 `[1,2,3,4]`

**예시 2**
- 입력: `nums = [0,3,7,2,5,8,4,6,0,1]`
- 출력: `9`
- 이유: 가장 긴 연속 수열은 `[0,1,2,3,4,5,6,7,8]`

---

## 📋 제약 조건 (Constraints)

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`

---

## 💡 접근 방식

### **HashSet으로 “수열의 시작점”만 확장하기**

정렬 없이 O(n)을 만들기 위해서는  
모든 숫자에 대해 무작정 좌우로 확장하면 안 되고,  
**“연속 수열의 시작점”인 숫자에서만 확장**해야 한다.

### 시작점 판단 규칙

어떤 숫자 `x`가 있을 때,
- `x - 1`이 Set에 **없다면**, `x`는 연속 수열의 **시작점**이다.
- 시작점일 때만 `x+1, x+2, ...`를 Set으로 확인하며 길이를 늘린다.

이렇게 하면 각 숫자는 “확장 과정에서” 최대 한 번씩만 방문되어  
전체가 O(n)에 가까워진다.
