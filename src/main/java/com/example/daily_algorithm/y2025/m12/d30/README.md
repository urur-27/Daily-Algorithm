# 🧩 Climbing Stairs

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** Dynamic Programming  
**링크:** https://leetcode.com/problems/climbing-stairs/

---

## 📝 문제 설명

계단이 `n`개 있다.  
한 번에 **1칸 또는 2칸**씩 오를 수 있을 때,  
정상까지 올라가는 **서로 다른 방법의 수**를 구하라.

---

## 🔍 예시

입력  
n = 2

출력  
2

설명  
1 + 1  
2

---

입력  
n = 3

출력  
3

설명  
1 + 1 + 1  
1 + 2  
2 + 1

---

## 📋 제약 조건 (Constraints)

- `1 <= n <= 45`

---

## 💡 접근 방식

이 문제의 핵심은 **현재 칸에 도달하는 방법은 이전 두 칸에서 온 경우의 합**이라는 점이다.


- `n`번째 칸에 도달하려면
    - `n-1`번째 칸에서 1칸 올라오거나
    - `n-2`번째 칸에서 2칸 올라온다

즉,

    dp[n] = dp[n-1] + dp[n-2]

이 식은 **피보나치 수열과 동일한 구조**를 가진다.
