# 🧩 Plus One

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** Array, Math  
**링크:** https://leetcode.com/problems/plus-one/

---

## 📝 문제 설명

정수 배열 `digits`가 주어진다.  
각 원소는 한 자리 숫자이며, 배열 전체는 **음이 아닌 정수 하나**를 표현한다.

이 정수에 **1을 더한 결과**를 배열 형태로 반환하라.

---

## 🔍 예시

입력  
digits = [1,2,3]

출력  
[1,2,4]

설명  
123 + 1 = 124 이다.

---

입력  
digits = [9,9]

출력  
[1,0,0]

설명  
99 + 1 = 100 이다.

---

## 📋 제약 조건 (Constraints)

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- 배열은 선행 0을 포함하지 않는다 (단, 숫자 0 자체는 예외)

---

## 💡 접근 방식

- 배열의 맨 뒤부터 탐색한다.
- 값이 9보다 작으면 1을 더하고 종료한다.
- 값이 9라면 0으로 바꾸고 올림을 다음 자리로 전달한다.
- 모든 자리가 9인 경우, 맨 앞에 1을 추가한 새 배열을 반환한다.
