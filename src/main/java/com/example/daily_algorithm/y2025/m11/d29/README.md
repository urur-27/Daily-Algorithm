# 🧩 205. Isomorphic Strings

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** HashMap, String  
**링크:** https://leetcode.com/problems/isomorphic-strings/
---

## 📝 문제 설명

두 문자열 `s`와 `t`가 주어졌을 때,  
`s`의 각 문자를 다른 문자로 치환하여 문자열 `t`를 만들 수 있다면  
두 문자열은 **isomorphic(동형)** 이라고 한다.

조건은 다음과 같다:

1. `s`의 각 문자는 **항상 동일한 문자로 매핑**되어야 한다. (일관성)
2. 서로 다른 문자가 **같은 문자로 매핑될 수 없다.** (충돌 방지)
3. 문자 간 매핑은 **1:1 대응**이어야 한다.

즉, 문자 패턴 구조가 동일한지 확인하는 문제이다.

---

## 🔍 예시

### ✔ 예시 1
입력

    s = "egg", t = "add"

출력

    true

설명:  
e → a  
g → d  
문자 매핑이 일관적이며 충돌 없음.

---

### ✔ 예시 2
입력

    s = "foo", t = "bar"

출력

    false

설명:  
f → b  
o → a 이어야 하는데,  
두 번째 o에서 a가 아니라 r 이어야 해서 일관성 실패.

---

### ✔ 예시 3
입력

    s = "paper", t = "title"

출력

    true

---

## 📋 제약 조건 (Constraints)

- `1 ≤ s.length, t.length ≤ 5 * 10⁴`
- `s`와 `t`는 ASCII 문자로 구성된다.

---

## 💡 접근 방식

문자 → 패턴 번호(Integer)로 인코딩해서 비교

- 문자열 `s`와 `t` 각각에 대해  
  **문자 → 패턴 번호(Integer)** 형태로 매핑한다.
- 예를 들어:
    - s = "foo" → 패턴: [1, 2, 2]
    - t = "bar" → 패턴: [1, 2, 3]
- 최종적으로 두 패턴 배열이 동일한지 비교하면 된다.
