# 🧩 283. Move Zeros

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** Two Pointers / Array  
**링크:** [283. Move Zeroes](https://leetcode.com/problems/move-zeroes/)
---

## 📝 문제 설명

정수 배열 `nums`가 주어졌을 때,  
모든 **0을 배열의 끝으로 이동**시키되,  
**0이 아닌 원소들의 상대적인 순서는 유지**하도록 배열을 재배치하라.

단, **새로운 배열을 생성하지 않고(in-place)** 문제를 해결해야 한다.

---

## 🔍 예시

**예시 1**  
입력:  
nums = [0, 1, 0, 3, 12]  
출력:  
[1, 3, 12, 0, 0]

---

**예시 2**  
입력:  
nums = [0]  
출력:  
[0]

---

## 📋 제약 조건 (Constraints)

- 1 <= nums.length <= 10⁴
- -2³¹ <= nums[i] <= 2³¹ - 1



---

## 💡 접근 방식

0을 뒤로 보내되,
0이 아닌 숫자들의 기존 순서는 그대로 지키는 것이 핵심

"0을 발견하면 뒤의 값과 교환" 같은 방식은
순서가 바뀌어버리기 때문에 사용할 수 없다.

대신 아주 단순하게:

앞에서부터 보면서 0이 아닌 숫자만 앞쪽에 차곡차곡 모은다.
→ 자연스럽게 0이 아닌 값들이 순서대로 앞으로 이동

앞으로 모으고 남은 나머지 공간은 전부 0으로 채운다.
→ 결과적으로 모든 0이 뒤로 감

배열을 복사하지 않고 같은 배열 안에서 해결하므로
in-place 방식으로 문제를 충족시킨다.
