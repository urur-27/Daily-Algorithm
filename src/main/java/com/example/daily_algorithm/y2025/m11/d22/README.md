# 🧩 21. Merge Two Sorted Lists 

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** 
**링크:** 
---

## 📝 문제 설명
정렬된 두 개의 링크드 리스트 `list1`, `list2`의 head가 주어진다.  
두 리스트의 노드들을 **이어 붙여(splice)** 하나의 정렬된 리스트로 병합한 뒤,  
그 **merged 리스트의 head**를 반환하는 문제이다.

두 리스트는 **오름차순(Non-decreasing order)** 으로 정렬되어 있다.


---

## 🔍 예시
### 예시 1
입력: 
```
list1 = [1,2,4]
list2 = [1,3,4]
```

출력: 
```
[1,1,2,3,4,4]
```

### 예시 2
입력:
```
list1 = []
list2 = []
```
출력:
```
[]
```

### 예시 3
입력:
```
list1 = []
list2 = [0]
```

출력:
```
[0]
```

---

## 📋 제약 조건 (Constraints)
- 각 리스트의 노드 수: `0 ≤ n ≤ 50`
- 노드 값: `-100 ≤ Node.val ≤ 100`
- 두 리스트는 모두 **오름차순으로 정렬되어 있음**


---

## 💡 접근 방식

노드값 반환을 위해 더미 노드 생성
더 작은 값의 노드를 붙이고 포인터를 이동
두 노드 중 한 노드의 비교가 끝나면, 나머지는 그대로 붙이기 (애초에 오름차순이니까)
