# 🧩 203. Remove Linked List Elements

**난이도:** Easy  
**플랫폼:** LeetCode  
**유형:** Linked List, Dummy Node  
**링크:** https://leetcode.com/problems/remove-linked-list-elements/  
---

## 📝 문제 설명
단일 연결 리스트의 head와 정수 `val`이 주어진다.  
리스트에서 **Node.val == val** 인 모든 노드를 제거하고  
변경된 리스트의 **새로운 head**를 반환하는 문제이다.

삭제해야 하는 값이 여러 개일 수 있으며,  
리스트의 중간・처음・끝 어디에 위치하더라도 모두 제거해야 한다.

---

## 🔍 예시

### ✔ 예시 1
**입력**  
```
head = [1,2,6,3,4,5,6], val = 6
```

**출력**  
```
[1,2,3,4,5]
```

---

### ✔ 예시 2
**입력**  
```
head = [], val = 1
```

**출력**  
```
[]
```


---

### ✔ 예시 3
**입력**  
```
head = [7,7,7,7], val = 7
```

**출력**  
```
[]
```


---

## 📋 제약 조건 (Constraints)

- 노드 개수: `0 ≤ n ≤ 10^4`  
- 노드 값 범위: `1 ≤ Node.val ≤ 50`  
- 삭제할 값 `val`: `0 ≤ val ≤ 50`  

---

## 💡 접근 방식

더미 노드를 만들고
주어진 노드들을 탐색하면서 val이 list.val이 아닌 노드들을 저장하고 반환한다.
